ShikshaSathi — Hackathon Project

ShikshaSathi ("Education Companion") is an ed-tech web platform born from a hackathon challenge to make quality education more accessible. The idea was simple but powerful: many students in underserved communities lack a single, organised place to find learning resources — so ShikshaSathi fills that gap with a clean, intuitive interface for course discovery, curated study materials, and structured learning paths.

Built under hackathon time pressure, it showcases the ability to ideate, design, and ship a working product rapidly without sacrificing usability. The site is fully responsive and deployed live on Vercel, reflecting both technical execution and a genuine social mission. It's a project that sits at the intersection of code and compassion.

International Sports Karate Federation

A professional, full-stack website built for the International Sports Karate Federation — a real organisation requiring a robust digital presence. This isn't just a static brochure site; it's a genuinely interactive, multi-user platform backed by Firebase, handling live data, user authentication, and dynamic content updates in real time.

The platform caters to three distinct audiences: athletes tracking events and rankings, coaches managing their students, and federation admins overseeing the entire ecosystem. Features include user registration and login, live database interactions, and a content management layer — all wrapped in a bold sports-themed design. Deployed on Netlify for fast global delivery, this project represents Astha's first serious step into backend-integrated web development.

MindSpark

MindSpark is an ambitious quiz-based learning platform currently under active development — designed to flip the script on boring self-study. The core idea: learning sticks better when it feels like a game. MindSpark will let students test their knowledge across topics, earn streaks for consistency, track their progress visually, and receive personalised suggestions based on where they struggle.

Planned features include timed quiz modes, difficulty levels, subject-wise breakdowns, and a clean dashboard showing growth over time. MindSpark draws inspiration from Astha's own experience as a student — and her belief that the right tools can make any subject feel conquerable. Watch this space.
